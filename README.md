# Requisitos

- PHP 8+
- Composer
- Node.js + NPM

---

# Instalação

- git clone
- cd projeto

- cp .env.example .env
- composer install
- npm install

- php artisan key:generate

---

# Rodando o projeto

**Opçao 1 - Usando Laravel Herd**

- Coloque o projeto dentro da pasta monitorada pelo Herd (ex: ~/Herd)
- Acesse no navegador: http://tempoo.test

**Opção 2 - Sem Herd**

- php artisan serve
- npm run dev
- acesse: http://localhost:8000
